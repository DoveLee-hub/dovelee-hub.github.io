1.2 什么是CSS

Cascading Style Sheet 层叠级联样式表

CSS：表现（美化网页）

字体、颜色、边距、高度、宽度、背景图片、网页定位、网页浮动……

1.3 CSS发展史

  • CSS1.0

  • CSS2.0

    • DIV（自定义块）+CSS
    • 提出了HTML与CSS分离的概念 ，网页变得简单，SEO

  • CSS2.1

    • 浮动
    • 定位

  • CSS3.0

    • 圆角，阴影，动画等等
    • 存在浏览器兼容性问题

1.4 CSS优势

  1. 内容和表现分离
  2. 网页结果表现统一，可以实现复用
  3. 样式十分丰富
  4. 建议使用独立于html的css文件
  5. 利用SEO，容易被搜索引擎收录

1.5 CSS的三种导入方式

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <!--内部样式-->
    <style>
        h1{
            color: green;
          /*选择器每一行都要有分号结尾*/
        }
    </style>
    <!--外部样式
    通过link进行外部索引
    -->
    <link rel="stylesheet" href="css/style.css">

</head>
<body>
<!--行内样式：
在标签元素中，编写一个style属性，编写样式即可
-->
<h1 style="color: red">我是标题</h1>
</body>
</html>

注：选择器每一行都要有分号结尾

优先级：后写的会把之前写的覆盖掉，所以行内样式优先级最高，内部样式和外部样式看谁离“被修饰块”近

拓展：外部样式两种写法

  • 链接式（推荐）
 <link rel="stylesheet" href="css/style.css">

  • 导入式（不推荐）

    会导致网页先出现结构，再进行渲染

    是CSS 2.1特有的

<style>
  @import url("css/style.css");
<style>

2.选择器

作用：选择页面上的某一个元素或某一类元素

2.1基本选择器

  1. 标签选择器：选择一类标签

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
   
    <style>
        /*标签选择器，会选择到页面上的所有这个标签的元素*/
    h1{
      color: #00ffff;
    /*在IDE左侧颜色方框处，点击，即可选择更多颜色*/
    }
        p{
            font-size: 80px;
        }
    </style>
</head>
<body>
    <h1>标题1</h1>
    <h1>标题2</h1>
     <p>这是内容</p>
</body>
</html>

  2. 类选择器 class：选择所有class一致的标签，跨标签

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
   
    <style>
        /*类选择器的格式 .class的名称{}
           好处：可以多个标签归类，是同一个class,可以复用
        */
        .DoveLee{
            color: red;
        }
        .MrLee{
            color: green;
        }
    </style>
</head>
<body>
    <!--class名字不能为数字-->
    <h1 class="MrLee">标题1</h1>
    <h1 class="DoveLee">标题2</h1>
    <p class="MrLee">这是内容</p>
</body>
</html>

  3. Id选择器：ID名全局唯一

    <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
   
    <style>
        /*ID选择器格式    ID必须保证全局唯一且ID名不可为数字
            #id名称{}
        */
        #DoveLee{
            color:red;
        }
    </style>
</head>
<body>
    <h1 ID="DoveLee">标题1</h1>
    <h1 ID="MrLee">标题2</h1>
    <p>这是内容</p>
</body>
</html>

优先级：ID选择器>类选择器>标签选择器

2.2 层次选择器

  1. 后代选择器：选择全部后代

    /*后代选择器*/
        body p{
            background: red;
        }

  2. 子选择器：只选择往后一代

     /*子选择器*/
        body>p{
            background: red;
        }

  3. 相邻兄弟选择器：选择相邻同辈（选择向下第一个同辈）

    /*相邻兄弟选择器*/ 
.active+p{
            background: red;
        }

  4. 通用选择器：选择向下所有同辈元素

/*通用兄弟选择器*/
        .active~p{
            background: red;
        }
    </style>
  1. 测试代码

2.3 结构伪类选择器

伪类：条件

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>

    <style>
        /*选择ul第一个元素*/
        ul li:first-child{
            background: red;
        }
        /*选择ul最后一个元素*/
        ul li:last-child{
            background: green;
        }

        /*选中p1:定位到父元素，选择当前的第一个元素
        选择当前p元素的夫元素，选中父元素的第一个，并且是当前元素（p元素）才生效
				不区分类型:比如我在p1前加了h1，则它会判定h1不是p，所以不执行动作
        */
        p:nth-child(1){
            background: cyan;
        }

        /*选中父元素下的第一个P元素
				区分类型：加了h1也无妨，他会找到第一个p元素并执行动作
				*/
        p:nth-of-type(1){
            background: blue;
        }
    </style>
</head>
<body>
<h1>h1</h1>
<p>p1</p>
<p>p2</p>
<p>p3</p>
<ul>
    <li>li1</li>
    <li>li2</li>
    <li>li3</li>
</ul>
</body>
</html>

2.4 属性选择器（常用）

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
    <style>
        .demo a{
            float: left;
            /*横向排列*/
            display: block;
            /*变为块*/
            height: 50px;
            width: 50px;
            border-radius: 10px;
            /*圆角矩形*/
            background: cyan;
            /*背景颜色*/
            text-align: center;
            /*字体居中*/
            color: azure;
            /*字体颜色*/
            text-decoration: none;
            /*去掉下划线*/
            margin-right: 5px;
            /*向右移动*/
            font: bold 20px/50px Arial;
            /*字体位置，使其在正方形中央*/
        }

        /*存在ID属性的元素*/
        /*a[id]{*/
        /*    background: yellow;*/
        /*}*/

        /*ID属性为first的元素*/
        /*a[id="first"]{*/
        /*    background: yellow;*/
        /*}*/

        /*class中有link的元素*/
        /*a[class*=link]{*/
        /*    background: yellow;*/
        /*}*/

        /*选中href属性中以http开头的元素*/
        /*a[href^=http]{*/
        /*    background: yellow;*/
        /*}*/

        /*选中href属性中以pdf结尾的元素*/
        /*a[href$=pdf]{*/
        /*    background: yellow;*/
        /*}*/
    </style>
<body>
    <p class="demo">

        <a href="http://www.baidu.com" class="link item first" id="first">1</a>
        <a href="" class="link item active" target="_blank" title="test">2</a>
        <a href="../photo.png" class="link item">3</a>
        <a href="index.html" class="link item">4</a>
        <a href="abc" class="link item">5</a>
        <a href="/a.pdf" class="link item">6</a>
        <a href="/abc.pdf" class="link item">7</a>
        <a href="abc,doc" class="link item">8</a>
        <a href="abcd.doc" class="link item">9</a>
        <a href="../receive.jpg"class="link item last">10</a>
    </p>

</body>
</html>

注：[]中可填写正则表达式
