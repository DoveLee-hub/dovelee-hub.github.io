Markdown学习

常用快捷键

标题

一级标题 # 标题

二级标题则为两个##

三级到六级同理

###字体

** 内容 ** 则为粗体 如 **Hello world**

* 内容 * 则为斜体 如*Hello world*

三个\*则为斜体加粗 如 ***Hello World***

~~ 则表示划去 如 ~~HelloWorld~~

引用

HelloWorld

引用用>+空格

分割线

分割线为 —和***

图片

![图片名]()

其中地址可以填本地地址和网络地址

其中网络地址的获得：右键图片 审查元素 即可看到地址

超链接

[超链接名字] + (超链接地址)

如：[apple](https://www.apple.com/cn/)

列表

数字+.+空格 回车之后便可以自动出现有序列表

如：

  1. A
  2. B
  3. C

-+空格 也可使用无序列表

如：

  • A
  • B
  • C
