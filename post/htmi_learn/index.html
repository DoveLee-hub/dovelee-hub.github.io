HTML详解

HTML简介

  • HTML

    Hyper Text Markup Language（超文本标记语言）

    超文本包括：文字，图片，音频，视频，动画等

  • HTML5的优势

    • 浏览器厂商的支持
    • 市场需求
    • 跨平台

  • W3C标准

    World Wide Web Consor（万维网联盟）

    W3C标准包括

    • 结构化标准语言（HTML、XML）
    • 表现标准语言（CSS）
    • 行为标准（DOM、ECMAScript）

网页基本信息

<!--告诉浏览器我们要使用什么规范-->
<!DOCTYPE html>
<html lang="en">
<!--head标签代表网页头部-->
<head>
    <!--meta描述性标签。用来描述网站的一些信息-->
    <!--meta一般用来做SEO-->
    <meta charset="UTF-8">
    <meta name="keynames" content="Dove,Lee,HTML">
    <meta name="Description" content="我的第一个HTML文件">
    <!--title标签代表网页名-->
    <title>DoveLee</title>
</head>
<!--body标签代表网页主体-->
<body>
Hello World
</body>
</html>

注：command + / 可以快速注释

代码一定要写到每个标签对儿中间 比如网页主体要写到……之中 标签一定要成对儿且闭合

图片标签

<img src="path" alt="text" title="text" width="x" height="y"/>
   <!--  图片地址 图像的替代文字 鼠标悬停提示文字  宽度        高度 -->

path : 图片地址（相对地址（推荐使用），绝对地址）

../ 上一级目录

alt：图片文字 指图片加载不出来时，显示的文字

超链接标签及应用

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>链接标签学习</title>
</head>
<body>
<!--锚标记-->
<a name="top">顶部</a>
<br>

<!--a 标签
href ： 必填，表示要跳转到哪个页面
target ： 表示窗口在哪里打开 _blank 在新的网页打开 _self在当前网页打开
-->
<a href="https://www.baidu.com" target="_blank">点击我跳转到百度</a>
<br>
<a href="https://www.baidu.com">
<!--通过点击图片跳转链接-->
<!--    <img src="path" alt="text" title="text" width="x" height="y"/>-->
</a>

<!--锚链接
1.需要一个锚标记
2.跳转到标记
-->
<a href="#top">回到顶部</a>

<!--功能性链接
邮件链接：mailto
QQ链接：QQ推广中获取链接
-->
<a href="mailto:958055356@qq.com">点击联系我</a>
</body>
</html>

块元素和行内元素

块元素

  • 无论内容多少，该元素独占一行
  • p、h1~h6……

行内元素

  • 内容撑开宽度，左右都是行内元素的可以排在一行
  • a、strong、em……

网页结构分析

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>页面结构分析</title>
</head>
<body>
<!--
header 标题头部区域的内存（用于页面或页面中的一块区域）
footer 标记脚步区域的内容（用于整个页面或页面的一块区域）
section Web页面的一块独立区域
article 独立的文章内容
aside 相关内容或应用
nav 导航类辅助内容
-->
<header>
    <h2>网页头部</h2>
</header>

<section>
    <h2>网页主体</h2>
</section>

<footer>
    <h2>网页脚部</h2>
</footer>

</body>
</html>

iframe内联框架

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>iframe</title>
</head>
<body>
<!--iframe内联框架
name:框架标识名
src:地址
width：宽度
height：高度
-->
<!--直接内联-->
<iframe src="https://www.baidu.com" name="baidu" frameborder="0" height="800px" width="1500px"></iframe>

<!--通过跳转内联-->
<iframe src="" name="baidu" frameborder="0" height="800px" width="1500px"></iframe>
<a href="https://www.baidu.com" target="baidu">点击跳转</a>

</body>
</html>

表单

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>表单学习</title>
</head>
<body>

<h1>注册</h1>
<!--表单form
action（必填）:表单提交地址，可以是网站，也可以是一个请求处理地址
method（必填）：post，get 提交方式
    get方式提交：我们可以在URL中看到我们提交信息，不安全，高效
    post方式提交：比较安全，传输大文件
-->
<form action="1.FirstHTML.html" method="get">

    <!--文本输入框：input type="text"
    value="DoveLee" 初识默认值
    maxlength="8" 最长能写几个字符
    size="30" 文本框长度
    -->
    <p>名字：<input type="text" name="username" value="DoveLee" maxlength="8" size="30"></p>

    <!--密码框：input type="password"-->
    <p>密码：<input type="password" name="pwd"></p>

    <!--单选框标签
     input type="radio"
     value：单选框的值（必须指定value）
     name：表示"组" 只有组相同才能实现单选
    -->
    <p>性别：
        <input type="radio" value="boy" name="sex" checked>男
<!--        checked 默认选中-->
        <input type="radio" value="girl" name="sex">女
    </p>

    <!--多选框
    input type="checkbox"
    -->
    <p> 爱好
        <input type="checkbox" value="sleep" name="hobby">睡觉
        <input type="checkbox" value="code" name="hobby" checked>敲代码
        <input type="checkbox" value="chat" name="hobby">聊天
        <input type="checkbox" value="game" name="hobby">游戏
    </p>
    <!--按钮
    button：普通按钮
    image：图片按钮
    submit:提交按钮
    reset：重置按钮
    -->
    <p>
        <input type="button" name="btn1" value="click">
        <input type="image" src="../../photo.png">
        <!--通过图片进行提交-->
    </p>
    
    <!--下拉框，列表框
    -->
    <p>国家
        <select name="列表名称">
            <option value="China">中国</option>
            <option value="US">美国</option>
            <option value="Switzerland" selected>瑞士</option>
            <!--selected 默认选中-->
            <option value="India">印度</option>
        </select>
    </p>
    <!--文本域textarea
        name：文本域名字
        cols：行数
        rows：列数
    -->
    <p> 反馈：
        <textarea name="textarea" cols="50" rows="10"></textarea>
    </p>

    <!--文件域
    input type = "file" name="files"
    -->
    <p>
        <input type="file" name="files">
        <input type="button" value="上传" name="upload">
    </p>

    <!--邮件验证
    input type="email" name="email"
    -->
    <p>邮箱验证
        <input type="email" name="email">
    </p>

    <!--URL验证
    input type="url" name="url"
    -->
    <p>url验证
        <input type="url" name="url">
    </p>

    <!--数字验证
    input type="number" name="num"
    -->
    <p>商品数量
        <input type="number" name="num" max="100" min="0" step="10">
    </p>

    <!--滑块验证
    input type="range" name="voice"
    -->
    <p>音量
        <input type="range" name="voice" min="0" max="100">
    </p>

    <!--搜索框
    input type="search" name="search"
    -->
    <p>搜索
        <input type="search" name="search">
    </p>

    <p>
        <!--增强鼠标可用性-->
        <label for="mark"> 你点我试试</label>
        <input type="text" id="mark">
    </p>

    <!--提交和重置按钮-->
    <p>
        <input type="submit">
        <input type="reset" value="清空表单">
    </p>
</form>
</body>
</html>

注：input中基本都要写name

其他拓展：input中添加关键字

  • readonly 只读
  • hidden 隐藏
  • disable 禁用
  • placeholder 提示信息
  • required 非空判断
  • pattern 正则表达式验证 （正则表达式百度查就完了）
